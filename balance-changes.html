Beta

Death Blackjack Beta Notes

Death Blackjack is still in beta. These notes cover the current public beta build and are not counted as part of the official stable v0.3.3 release coverage.

Between-Run Home

  • New: Home page now acts as the between-runs hub with Continue/New Run flow, titles, curses, achievements, run logs, keybinds, and account/save utilities.
  • New: Progress Route now presents visible level rewards and unlock pacing through a longer House progression ladder.
  • New: title loadout supports up to 3 equipped titles at once.
  • New: achievements and title goals now show clearer visible progress on the home screen.

New Cards

  • Ace Anchor: Ace hands win more and lose less.
  • Grim Chip: strong win payout with added loss pressure.
  • Dealer Trap: rewards dealer busts harder.
  • Finisher Card: pays extra on 20 or 21.

New Passives

  • Boss Dividend: boss round wins pay extra cash.
  • Vault Interest: Casino Vault nodes pay extra cash.
  • Trial Script: Skill Trial wins pay extra cash.
  • Croupier Stash: start each table with Lucky Draw charges.

Meta Additions

  • New: titles now carry rarities and deck-shaping effects.
  • New: House curses can be equipped after a clear to make future runs harder in exchange for more progression score.
  • New: save/account tools now support export, join, leave, and between-device transfer from the home page.

Table and Route Updates

  • New: the World 1 play page now uses a separate table page with cleaner startup flow and less chance of getting trapped in bad saved states.
  • Updated: the map overlay, forge flow, dealer summaries, and route panels were rebuilt to read more clearly during a run.
  • Updated: more cards, passives, progression rewards, and title unlocks are now visible in the beta build.
Game 01

Gambling King Live Details

Upgrade Additions

  • New: Fee Broker: 0 -> 12% -> 24% -> 36% fee reduction across levels 0 to 3.
  • New: Safety Net: 0% -> 12% -> 24% -> 36% dead-spin refund across levels 0 to 3.
  • New: Purchase caps: repeated buying of strong Gambling King upgrades now stops at fixed limits.
  • New: House Bribe: buys 10 spins of lower fee and debt-interest pressure for rich late-game runs.
  • New: Overclock Spin: stores one premium spin with stronger payout, wild odds, and proc chances.
  • New: Emergency Buyout: expensive bailout that cuts debt and can erase a currently due table fee.
  • New: VIP Table Pass: stores future half-price table fee payments.

Late-Run Access

  • These cash sinks are locked until the run is clearly in late-game territory.
  • Current unlock paths: max 4 capped purchases, reach best net worth $4500, or survive 60 spins with 5+ reels.
  • Reason: late-run money sinks should help rich survival-focused runs without flattening the early game.

Hell Target

  • Hell is meant to be the hardest core difficulty before Gods Will.
  • The goal was to keep it brutal without making it feel impossible.

Hell Exclusive Rules

  • New: Hell payout factor: 1.00 -> 0.70 on the Hell-only multiplier layer.
  • New: even after the global payout easing, Hell keeps a harsher reward rule than other difficulties.

Gods Will

  • New: Gods Will difficulty added above Hell.
  • It is intended to be the hardest mode in the game.

Gods Will Exclusive Rules

  • New: Gods Will payout factor: uses x0.50 on the Gods Will-only multiplier layer.
  • New: Gods Will fee factor: uses x1.18 on table fees.
  • New: Gods Will interest rule: adds +1.8% debt interest.
  • New: Gods Will win target: needs 135 spins to clear.
  • New: Gods Will debt limit rule: reduces effective debt limit by 250.
  • New: Gods Will wild rule: cuts wild generation to x0.5.
  • New: Gods Will reel expansion rule: uses 0.45 extra-slot payout penalty per slot.

Gods Will Difficulty Data

  • Gods Will: Start money 700, debt limit 860, fee base 72, fee growth 1.28, payout scale 1.13, devil chance 12.8%.

Difficulty Changes

  • Hard: start money 760 -> 670, debt limit 1600 -> 1080, fee base 58 -> 69, payout scale 1.20 -> 1.12.
  • Expert: start money 730 -> 640, debt limit 1450 -> 900, fee base 62 -> 74, payout scale 1.18 -> 1.10.
  • Master: start money 700 -> 620, debt limit 1300 -> 820, fee base 66 -> 77, payout scale 1.16 -> 1.09.
  • Nightmare: start money 670 -> 600, debt limit 1180 -> 760, fee base 70 -> 80, payout scale 1.14 -> 1.08.
  • Insane: start money 640 -> 590, debt limit 1060 -> 700, fee base 74 -> 82, payout scale 1.12 -> 1.07.
  • Hell: start money 610 -> 700, debt limit 428 -> 860, fee base 78 -> 72, payout scale 1.10 -> 1.13.
Game 02

Death Wheel Live Details

New Systems

  • New: Lucky Charm added to the shop. Increases Lucky slot chance from 0 to 20% across upgrade levels.
  • New: Curse Cleanse added to the shop. Removes 1 active curse, or gives +1 shield if none are active.
  • New: Heart Boost added to the shop. Restores +1 life up to the run maximum.
  • New: Purchase caps were added to stronger Death Wheel shop items.
  • New: Dead Heat chaos event. Next 4 spins use stronger Lucky rewards and stronger Death penalties.
  • New: Crooked Dealer chaos event. Next 6 spins temporarily increase both Lucky and Death odds.
  • New: Flash Market chaos event. Opens an immediate high-price shop with fewer offers.
  • New: Last Breath Lucky event. Low-resource rescue that grants life, shield, and cash.
  • New: Treasure Pocket Lucky event. Lucky can open a free shop pickup instead of only raw money.

Removed

  • Removed: Slot Freeze left the shop pool because it overlapped too much with Prediction Chip and Slot Lock.

Core Changes

  • Start money: 1200 -> 1800, spin cost: 25 -> 20, start lives: 3 -> 6.
  • Scheduled shop cadence: 6 spins -> 4 spins, critical scaling cadence: 10 spins -> 14 spins.
  • Death penalty: max(150, 25%) -> max(125, 18%), instant death: 5% -> 0%.
  • Lucky rewards: 75/125/200 -> 90/145/225.

Wheel Layout

  • Normal slots: 16 -> 17
  • Death slots: 4 -> 3
  • Shield slots: 2 -> 3
  • Other slot counts stayed the same.
