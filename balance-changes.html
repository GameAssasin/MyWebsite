Website v0.2 - Gambling King Changes

Format below shows exactly what changed from website v0.1 to website v0.2.

Core Economy Changes (v0.1 -> v0.2)

  • Global payout ease multiplier: 1.00 -> 2.00.
  • Hell-only payout factor: 1.00 -> 0.70.
  • Effective Hell payout multiplier vs pre-v0.2 formula: 1.00 -> 1.40.
  • Fixed spin cost: $115 -> $115 (unchanged after testing).

Difficulty Start Cash Changes (v0.1 -> v0.2)

  • Hard: $553 -> $760.
  • Expert: $479 -> $730.
  • Master: $291 -> $700.
  • Nightmare: $245 -> $670.
  • Insane: $199 -> $640.
  • Hell: $168 -> $610.

Hard+ difficulties were fully retuned in v0.2 (cash, debt limits, interest, fee curves, payout scales, devil rates).

Why This Changed

  • Make all difficulties easier overall.
  • Keep Hell challenging instead of free after the global buff.
