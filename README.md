# MyWebsite

Personal website project with browser mini-games.

## Project Path
`C:\Users\paul\Documents\MyWebsite`

## Pages
- `index.html` - Home
- `about.html` - About
- `rules.html` - Help and rules
- `gambling-king.html` - Gambling King game
- `death-wheel.html` - Death Wheel game

## Gambling King (Current)
- Difficulty system with 10 modes (`Noob` to `Hell`).
- Inventory upgrades and shop items.
- Reel Expansion system (up to 8 slots).
- Bigger combo payouts for 4x/5x/6x+ matches.
- Unsaved roguelite run flow (refresh resets run).

## Key Files
- `gambling-king.js` - Game logic and balance values.
- `gambling-king.html` - Game UI layout.
- `styles.css` - Shared styles.
- `death-wheel.js` - Death Wheel logic.

## Run / Test
Open `index.html` directly in a browser, or run any static file server from this folder.
